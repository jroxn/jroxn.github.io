CentOS8安装之后的一些配置

linux 系统 centos8
system
发布日期:   2020-05-19
更新日期:   2021-08-10
文章字数:   624
阅读次数:  

网卡固定IP

  修改网卡信息 

TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="none"   # 设置静态
DEFROUTE="yes"
IPV4_FAILURE_FATAL="yes"
IPV6INIT="yes"
IPV6_AUTOCONF="yes"
IPV6_DEFROUTE="yes"
IPV6_FAILURE_FATAL="no"
IPV6_ADDR_GEN_MODE="stable-privacy"
NAME="ens160"
UUID="1f22ea44-481f-410e-b062-7b05fa7a8375"   # 设备唯一标识，不要更改自己的UUID!!!
DEVICE="ens160"   # 网卡名，不要更改自己的!!!
ONBOOT="yes"   # 开机自动加载网卡
HWADDR="00:0C:29:D5:D6:DE"   # 唯一MAC地址，不要更改自己的!!!
IPADDR="192.168.1.221"   # 设置一个固定IP
PREFIX="24"   # 掩码长度 另一种写法NETMASK="255.255.255.0"
GATEWAY="192.168.1.1"   # 网关（路由器、交换机IP）
DNS1="114.114.114.114"　　# DNS地址，开机默认加载
IPV6_PRIVACY="no"

  使配置生效 

ifdown ens160 && ifup ens160   # 执行完之后重新连接

profile

  编辑/etc/profile文件最后添加vi别名 

alias vi=vim

vimrc

  编辑/etc/vimrc文件最后加入 

" 设置tab建4个空格
set ts=4
set expandtab
set autoindent
" 设置vim显示行号
set nu
" 设置F9粘贴缩进，默认原格式粘贴，按F9自动缩进粘贴
set pastetoggle=<F9>
set paste

补充：

  修改已保存的文件

  1. Tab 键替换为空格 
    :set ts=4
:set expandtab
:%retab!
  2. 空格替换为Tab 
    :set ts=4
:set noexpandtab
:%retab!

冷知识：加感叹号!是用于处理非空白字符之后的Tab，即所有的Tab；不加，则只处理行首的Tab。
  好冷~ ≡(▔﹏▔)≡ 🥶

yum

  这里配置的是aliyun的yum源

  1. CentOS-Base.repo仓库 
cd /etc/yum.repos.d/
mkdir old
mv *.repo old/
curl -o /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-8.repo
# 非阿里云ECS用户会出现 Couldn't resolve host 'mirrors.cloud.aliyuncs.com' 信息，不影响使用。可自行修改相关配置: 
sed -i -e '/mirrors.cloud.aliyuncs.com/d' -e '/mirrors.aliyuncs.com/d' /etc/yum.repos.d/CentOS-Base.repo
  1. epel(RHEL 8)软件仓库

安装 epel 配置包 

yum install -y https://mirrors.aliyun.com/epel/epel-release-latest-8.noarch.rpm

将 repo 配置中的地址替换为阿里云镜像站地址 

sed -i 's|^#baseurl=https://download.fedoraproject.org/pub|baseurl=https://mirrors.aliyun.com|' /etc/yum.repos.d/epel*
sed -i 's|^metalink|#metalink|' /etc/yum.repos.d/epel*

firewall

  关闭firewalld 

systemctl stop firewalld
systemctl disable firewalld

selinux

  关闭selinux

sed -i.bak 's/enforcing$/disabled/g' /etc/selinux/config
setenforce 0   # 临时关闭，重启失效

软件安装

yum -y install wget vim 
yum update -y   # 软件升级

配置好之后重启

reboot
source /etc/profile   # 不想重启的话使用此，使配置文件生效
