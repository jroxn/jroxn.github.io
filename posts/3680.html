需求：

  hexo搭建个人网站，使用OSS+CDN加速。（涉及到域名备案（又涉及到服务器购买）、因为购买了ECS打算利用一下做Git仓库，做web访问）

阿里云备案必花钱！备案免费，服务器要钱具体看下图：

买了一个最便宜的服务器102元/年，好像有99元/年的，域名wujiops.cn首年29元/年，续费39元/年

备案成本：102+29=131元

以后花费：39+OSS+CDN≈100左右，预算超过300元/年，打算用GitHub pages+免费CDN，年费0元。

接着介绍git仓库搭建情况，及上线

系统情况：

# 系统版本
[root@www opt]# cat /etc/redhat-release 
CentOS Linux release 8.1.1911 (Core) 

# yum源
[root@www yum.repos.d]# cat CentOS-Base.repo 
# ...注释省略
[BaseOS]
name=CentOS-$releasever - Base
baseurl=http://mirrors.cloud.aliyuncs.com/$contentdir/$releasever/BaseOS/$basearch/os/
gpgcheck=1
enabled=1
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-centosofficial

# epel源
[root@www yum.repos.d]# cat CentOS-epel.repo 
[epel]
name=Extra Packages for Enterprise Linux 8 - $basearch
baseurl=http://mirrors.cloud.aliyuncs.com/epel/8/Everything/$basearch
failovermethod=priority
enabled=1
gpgcheck=0
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-EPEL-8

软件安装

# 均为yum安装
[root@www ~]# yum -y install httpd vim git
[root@www ~]# systemctl enable httpd
[root@www ~]# systemctl start httpd

git仓库

[root@www ~]# cd /opt/
[root@www opt]# git init --bare wujiops.git
[root@www opt]# ls
wujiops.git

报错

git commit -m "1"
#    git config --global --edit
#    git config --global user.name "Your Name"
#    git config --global user.email you@example.com
git config --global --edit   # 执行，设置全局配置

# 或者执行下面两个命令
# git config --global user.name "Your Name"   # 提交者姓名
# git config --global user.email you@example.com   # 提交者邮箱

本地仓库连接远程仓库

git remote add origin https://xxx.git   # ssh://xxx.git，有些仓库没有.git后缀

SSH免密登录


  指明ECS服务器IP地址和端口 

$ ssh-copy-id root@*.*.*.*: -p ***

本地上传

repository: ssh://root@*.*.*.*:***/opt/wujiops.git

服务器获取

[root@www www]# rm -fr html
[root@www www]# git clone ssh://root@localhost:xxx/opt/wujiops.git html
[root@www www]# cd html/
[root@www html]# ls
404.html  archives  categories  err          friends  index.html  lanmp  medias  other   page    search.xml  wu-ji-ops
about     atom.xml  css         favicon.png  hadoop   js          libs   notes   others  python  tags
[root@www html]# git pull
Already up to date.
