ubuntu 18 安装

约定

  VMware 版本 15.5.1 build-15018445
  Ubuntu18 版本 ubuntu-18.04.3-live-server-amd64

Ubuntu18安装

新建虚拟机,选择典型

选择下载好的Ubuntu18-server镜像

个性化(用户名密码)

安装位置,和虚拟机名称

磁盘选择20G,存储为单个文件

点击完成

开机设置,设置使用英语

键盘默认,不用改

选定网卡,选择IPv4

设置Manual

填入自己的IP,网关等.

我的如下

查询自己的信息

  在VMware编辑里面,选择NAT模式,进入NAT设置,查看网关信息.依据此,填写自己的IP和网关.

填完效果如下

代理跳过,不设置

修改为阿里云apt源

选择安装磁盘

直接回车

分好的区

  20G磁盘没有必要自定义分区,服务器安装请勿参考此分区.

选择格式化

填入姓名,主机名,密码

选择安装OpenSSH

跳过,直接Done

等待安装,View full log可以查看安装日志

重启,完成安装

用户名,密码登录

会打印启动日志,覆盖掉登录窗口,直接回车显示登录信息

开机设置,root密码,SSH远程登录

# 设置root密码
wang@wujiops:~$ sudo passwd 
[sudo] password for wang:    # 当前用户密码
Enter new UNIX password:    # 新root密码
Retype new UNIX password:    # 再次输入
passwd: password updated successfully

wang@wujiops:~$ sudo vi /etc/ssh/sshd_config
# 揭开注释 & 修改以下两行内容
Port 22
PermitRootLogin yes

wang@wujiops:~$ sudo service sshd restart   # 重启sshd服务

  打开xshell尝试链接吧!
