例子

算术/赋值运算符

注意： Python中不存在变量的定义，变量在Python中就是一个名字。c = 1，即把1赋值给c

>>> c = 1
>>> c = c + 1
>>> print(c)
2
>>> c+=1   #相当于c = c + 1
>>> print(c)
3
>>> c++,c--   #++，--自增和自减运算符，（shell体现）但是在python中不能这样写
>>> b = 2
>>> a = 3
>>> b+=a   #相当于b = b + a
>>> print(b)
5
>>> b-=a   #相当于b = b - a
>>> print(b)
2
>>> b*=a   #相当于b = b * a
>>> print(b)
6

比较/逻辑运算符

数字，字符串，布尔值都可以比较，返回值是布尔值（==，!=，>，<，>=，<=）

>>> True > False
True
>>> 1==1  #==如果两个变量取值相等，返回结果True，参考下文对比is
True
>>> 1>1
False
>>> 'abc'=='abc'
True
>>> 1!=2
True
# b = 1，b+=b>=1，b的值
>>> b = 1
>>> b+=b>=1
>>> print(b)
2
>>> print(b>=1)
True
>>> b = b + True
>>> print(b)
3
>>> int(True)
1
# 字符串比较的是ascll码
>>> ord('a')   #ord()查看元素的ascll码
97
>>> ord('b')
98
>>> 'a' < 'b'
True
>>> [1,2,3] < [2,3,4]
True
>>> (1,2,3) < (2,3,4)
True

逻辑运算符

（and两者都为真才是True，or有一个为真就是True，not双not抵消）（或or、与and、非not）

>>> True and True
True
>>> False and False
False
>>> False and True
False
>>> False or True
True
>>> not True
False
>>> not not True
True
>>> 1 and 1   #相当于True and True
1
>>> 'a' and 'c'   #参考下文,理解：and  or
'c'
>>> not 'a'
False
>>> 'a' or 'b'
'a'
# 对于int整形和float浮点型，0表示False，非0表示True
# 对于str字符串类型，‘’空表示False，非空表示True
>>> not 0
True
>>> not ''
True
>>> not '0'
False
# 对于list列表，[]空列表是False，非空表示True
# 对于tuple元组，set集合，dict字典；同样遵循列表的风格。空表示False，非空表示True
>>> not {}
True
>>> not ()
True
>>> not dict()
True
# 理解：and  or
# 当计算机读到第一个数的时候，并不能给出结果。只有将全部的代码解析完，才能出结果。
>>> 'a' and 'b'   #非空都为True，但是会显示第二个（最后一个）代码（值）。
'b'
>>> 'b' and 'a'
'a'
>>> 0 and 'c'   #0为False
0
>>> 'c' and 0
0
>>> 'b' and 'a' and 'c'
'c'

现实例子： 1 > 0，计算机必须读到0的时候才会知道结果是True，读到1的时候不会有结果。

or运算

区别： 因为or是有一个为真就返回True，所以它只需要读到第一个真就返回结果就行了。（避免计算资源浪费😊）

>>> 0 or 1 or 3
1
>>> 3 or 1 or 0
3
>>> 0 or False or 'a' or 1
'a'

成员运算符

in，not in（list列表，tuple元组，str字符串，set集合，dict字典）

>>> a = 1
>>> a in [1,2,3]   #判断a是否在list列表[1,2,3]中
True
>>> 4 in [1,3,4]
True
>>> b = 5
>>> a = [1,2,3]
>>> b in a
False
>>> a+=[4,5]
>>> b in a
True
>>> c+=b   #c = c + b，print(c) = 3
>>> c not in a   #c是否不在list列表a中
True
>>> print(b)
5
>>> print(c)
8
>>> b = 'a'   #字典dict
>>> b in {'c':1}
False
>>> b = 'c'
>>> b in {'c':1}   #对于字典dict来说，成员运算符是判断的key:value中的key
True
>>> b = 1
>>> b in {'c':1}
False

身份运算符

比较的是两个变量的身份是否相等（和==有区别，is比较的是内存地址）

>>> 'a' is 'a'
True
>>> 1 is 1
True
>>> 1 == 1.0
True
>>> 1 is 1.0
False
>>> id(1.0)
1270122593352
>>> id(1)
140728377594912
>>> [1,2] is [1,2]   # 这里是两个列表，所以显示False，记作lista 和 listb
False
>>> id([1,2])   # 这里是listb
1270130542216
>>> id([1,2])   # 这两次查询都是最后一个列表内存地址，这里也是listb
1270130542216
>>> [1,2] is [1,2]   # 这里变成了listc 和 listd
False
>>> id([1,2])   # 这里是listd的内存地址
1270133275656
# a = {1,2,3}，b = {2,1,3}；求a == b，a is b。
>>> a = {1,2,3}
>>> b = {2,1,3}
>>> a == b   # 集合set和字典dict是无序序列，所以顺序变化不影响取值（无序序列：dict{}字典，set{}集合，有序序列[]lise列表，()tuple元组，''str字符串）
True
>>> {'a':1,'b':2} == {'b':2,'a':1}
True
>>> a is b
False
>>> id(b)
1270133037096
>>> b is b
True
>>> c = [1,2,3]
>>> c is c   # 变量赋值 比较的是 同一个变量，所以返回True
True
>>> c is [1,2,3]   # 相同的集合set{}，dict字典{}，list列表[]，直接比较返回False，因为身份运算比较的内存地址
False
>>> [1,2]   # 验证两个[1,2]列表，内存地址是不同的。第二个[1,2]相当于一个新的list不是原来的了
[1, 2]
>>> id([1,2])
1270130541064
>>> [1,2]
[1, 2]
>>> id([1,2])
1270130542216

type类型

type类型判断

>>> a = 'hello'   # 把判断a的类型是不是str，返回True
True
>>> isinstance(a,str)   # 判断a是不是str类型
True
>>> isinstance(a,(int,str,tuple))   # 判断a是否属于元组tuple()中的其中一个
True
>>> isinstance(a,(int,tuple))
False

对象的三个特征： id、value、type；
对应判断方式： is（判断id）、==（判断value）、isinstance（判断type），（type类型判断，不能判断对象的子类型属于某一种类型，isinstance可以）
位运算符： &按位与、|按位或、^按位异或、~按位取反、<<左移动、>>右 移动

>>> a = 2
>>> b = 3
>>> a & b
2
>>> bin(a)
'0b10'
>>> bin(b)
'0b11'

与IP地址划分作类比。按位与&，只有 同位值都为1 结果才是1。

按位或|，只要 同位值有一个为1 结果就是1。
