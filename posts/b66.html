sourcetree3.x 跳过注册bitbucket

学习笔记 git sourcetree
others
发布日期:   2020-06-01
更新日期:   2021-08-10
文章字数:   661
阅读次数:  

  前提：使用腾讯电脑管家自带的软件管理工具，先安装git，在安装sourcetree，这两个软件在软件管理里面都有，安装git不用修改配置，直接下一步，直到安装完成，这个时候再安装sourcetree。如果你是菜鸟sourcetree全中文界面可以很好的带你入门git，一些错误仍需要您手动执行git命令，很少情况下会发生，如果你是用git的老手，sourcetree可以让您的工作效率事半功倍🏊‍。

测试jsdelivr加速GitHub图片，受jsdelivr服务器影响，可能会造成图片显示异常

使用软件管理安装sourcetree

第一步直接关闭此窗口。

使用+R键，输入%LocalAppData%\Atlassian回车，接着进入SourceTree目录，创建accounts.json文件： 

[
  {
    "$id": "1",
    "$type": "SourceTree.Api.Host.Identity.Model.IdentityAccount, SourceTree.Api.Host.Identity",
    "Authenticate": true,
    "HostInstance": {
      "$id": "2",
      "$type": "SourceTree.Host.Atlassianaccount.AtlassianAccountInstance, SourceTree.Host.AtlassianAccount",
      "Host": {
        "$id": "3",
        "$type": "SourceTree.Host.Atlassianaccount.AtlassianAccountHost, SourceTree.Host.AtlassianAccount",
        "Id": "atlassian account"
      },
      "BaseUrl": "https://id.atlassian.com/"
    },
    "Credentials": {
      "$id": "4",
      "$type": "SourceTree.Model.BasicAuthCredentials, SourceTree.Api.Account",
      "Username": "",
      "Email": null
    },
    "IsDefault": false
  }
]

使用+R键，输入%LocalAppData%\Atlassian回车，接着进入SourceTree.exe_Url_czfpfghshbirstykwmn1gvqek200yvfx目录，此目录名会随着版本变化，接着进入3.3.8.3848目录，打开user.config文件，在里面加入六行代码。我的是3.3.8，而腾讯软件管理的版本是3.2.6，可在”工具T”–>”选项O”中，关闭updates，禁用更新检查。
放入<SourceTree.Properties.Settings>...</SourceTree.Properties.Settings>标签中即可 

<setting name="AgreedToEULA" serializeAs="String">
    <value>True</value>
</setting>
<setting name="AgreedToEULAVersion" serializeAs="String">
    <value>20160201</value>
</setting>

在软件管理中打开sourcetree

提示未找到Mercurial

选择不想使用Mercurial

这里是选择了第一个选项。

跳过Mercurial或者下载，完成后就进入使用界面了

配置git和openssh

git安装完成后，在桌面右键打开git gui，选择show ssh key

选择Generate Key

连续点两次yes


完成之后会在当前用户家目录生成公钥和私钥

这里的公钥长期存在，您可以复制到剪切板Copy To Clipboard，粘贴到GitHub、coding等SSH公钥位置，从而进行免密git push origin branch

在”工具” “选项”中，选择openssh即可

这个时候你就可以把状态栏的”戴蓝帽子的小电脑”干掉了，点击”ssh密钥”后面的”…”可以进入.ssh目录查看自己的私钥id_rsa和公钥id_rsa.pub
